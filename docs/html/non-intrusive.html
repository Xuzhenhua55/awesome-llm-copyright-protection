Non-intrusive Fingerprinting

Non-intrusive fingerprinting methods leverage the inherent properties of language models without requiring modifications to their architecture or training process.

Weight Space Based

Methods that analyze the weight space of language models to identify unique patterns and characteristics.

Feature Space Based

Techniques that utilize different types of features extracted from language models for fingerprinting.

Activation Features

Methods that analyze the activation patterns of neurons in the model to create unique fingerprints.

Logits Features

Approaches that utilize the output logits distribution to create distinctive fingerprints.

Semantic Features

Techniques that leverage semantic representations and embeddings for model fingerprinting.

Prompt Optimization Based

Methods that use carefully crafted prompts to elicit distinctive responses that reveal model identity.
